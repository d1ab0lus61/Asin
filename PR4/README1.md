### SimpleAsyncArrayExample - Опис програми

#### Опис
Програма **SimpleAsyncArrayExample** демонструє використання асинхронних операцій для генерації та обробки масивів у Java за допомогою класу `CompletableFuture`. Вона створює двовимірний масив цілих чисел (3x3) та виконує кілька операцій: генерує випадкові значення для елементів масиву, виводить початковий масив, а також виводить кожен стовпець масиву окремо.

Основною ідеєю цієї програми є:
1. Використання асинхронних завдань для зменшення часу очікування під час виконання обчислень.
2. Демонстрація ефективності роботи з багатозадачністю в Java за допомогою `CompletableFuture`.
3. Оцінка часу, що витрачається на кожну операцію за допомогою вимірювань часу виконання.

#### Основні етапи роботи програми
1. **Генерація випадкових чисел**: Програма генерує двовимірний масив розміру 3x3, заповнюючи кожен елемент випадковим числом у діапазоні від 0 до 99.

2. **Виведення початкового масиву**: Після генерації масиву програма виводить його на екран.

3. **Виведення стовпців масиву**: Програма по черзі виводить значення кожного стовпця масиву.

4. **Оцінка часу**: Для кожної з операцій вимірюється час виконання, що дозволяє оцінити ефективність асинхронного підходу.

#### Ключові моменти програми
- **Асинхронність**: Програма використовує методи `thenAcceptAsync` та `thenRunAsync`, щоб виконувати обробку масиву в окремих асинхронних потоках.
- **Виведення результатів**: Час, що затрачається на кожну операцію, вимірюється у наносекундах і виводиться в мілісекундах.
- **Обробка стовпців**: Окремо обробляються та виводяться стовпці масиву.

#### Висновки
Програма є простим прикладом використання асинхронного програмування в Java. Вона допомагає зрозуміти, як можна виконувати кілька незалежних задач паралельно, використовуючи можливості сучасного Java.
